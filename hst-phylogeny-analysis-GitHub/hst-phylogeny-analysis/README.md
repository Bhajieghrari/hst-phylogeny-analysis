## 📖 Overview

This repository contains all custom Python scripts and analysis pipelines used in the manuscript:

**"Molecular Evolution and Phylogeny of the Plant HASTY Family, the Plant Orthologs of Exportin-5"**

*Behzad Hajieghrari, Hadiseh Salahi, Mojahed Kamalizadeh*

### What this repository does:
- ✅ Sequence curation and quality control
- ✅ Multiple sequence alignment with 5 algorithms
- ✅ Motif discovery and analysis (MEME Suite)
- ✅ Phylogenetic inference (IQ-TREE)
- ✅ Structural modeling and comparison (MODELLER)
- ✅ Publication-ready figure generation
