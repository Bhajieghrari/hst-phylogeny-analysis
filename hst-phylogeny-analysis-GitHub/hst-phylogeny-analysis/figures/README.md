# Figures Directory

This directory is expected to contain the manuscript figures
(`Figure_1.png` … `Figure_9.png`, `Supplementary_Figure_S1.png` …
`Supplementary_Figure_S7.png`) as listed in
`docs/repository_structure.txt`.

**Note:** the original `Github.rar` archive did not include the actual
figure image files, only the scripts and documentation. Run
`scripts/figure_generation.py` against a populated `results/` directory
to regenerate them, or copy the final figure files here manually.
