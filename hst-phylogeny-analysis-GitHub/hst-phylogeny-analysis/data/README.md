# Data Directory

This directory contains the input data for the analysis.

## Files

- `curated_sequences.fasta` - Final curated dataset of 364 HST/XPO5 sequences
- `raw_sequences.fasta` - Original sequences from UniProt (before curation)

## Data Sources

All sequences were retrieved from UniProt (https://www.uniprot.org/) 
with taxonomy restricted to Viridiplantae.
